# Smart Document Search Engine

A simple search engine built in Python using the TF-IDF algorithm.

## Features
- Search through multiple documents
- Ranks results based on relevance
- Uses TF-IDF scoring
- Simple command line interface

## Technologies
- Python
- TF-IDF algorithm
- Text processing

## Project Structure
documents/
    chemistry.txt
    football.txt
    physics.txt
main.py

python main.py
